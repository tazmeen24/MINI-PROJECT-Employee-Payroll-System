# Employee Payroll System

The **Employee Payroll System** is a Java-based application designed and implemented using core **Object-Oriented Programming (OOP)** principles.

It manages employee payroll efficiently by supporting multiple employee types and automating salary-related processes.

The system includes features such as salary calculation based on employee type, automatic tax and provident fund deductions, and payslip generation.

## OOP Concepts Used

- **Abstraction** - Implemented using abstract classes (`Person`, `Employee`)
- **Encapsulation** - Private fields with controlled access through getters/setters
- **Inheritance** - `Employee` extends `Person`, and subclasses extend `Employee`
- **Polymorphism** - `calculatePay()` method is overridden for different employee types
